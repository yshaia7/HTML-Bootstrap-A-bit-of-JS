מטרות הקורס

כיצד בונים אתר כמו Facebook?

הקורס מקנה ידע בטכנולוגיות תכנות המשולבות באתרים מתקדמים ,עם דגש על חוויה אינטראקטיבית עשירה, תוכן דינמי, ופיתוח אתרים המתואמים לכל סוגי מכשיר (Responsive).

לשם בניית אתר שכזה נדרשת יכולת לעצב אתר בצורה תקינה, ידע בטכנולוגיות לפיתוח צד לקוח וצד השרת.

בצד הלקוח זה כולל: HTML5, CSS3 ,JavaScript וידע בספריות השייכות לחברות המובילות כגון Twitter Bootstrap .

בצד השרת זה כולל ידע בניהול מסדי נתונים ,הקמת שרת Web רב לקוחות, פיתוח רכיבים כגון רישום משתמשים, אימות, ניהול תוכן דינמי, חיפוש במאגרי מידע, ניהול סל קניות. נלמד לפתח על בסיס ארכיטקטורה Model-View-Controller‏ ‏ ע״י שימוש במערכות(front-end/backend frameworks) כגון Node.JS.

נושאי הקורס

  • בנית ועיצוב דפים ע”י HTML, CSS
  • Responsive design
  • תקשורת http
  • מבנה Document Object Model
  • Javascript (ECMASCript 6)
  • כלים לפיתוח וdebugging עבור אתרים משולבות שרת-לקוח
  • Asynchronous Programming (JSON, Promises, web workers)
  • אתרים בנויים על בסיס דף יחיד: Single Page Applications
  • טפסים, אימות(authentication)
  • ניהול מצבים בין דפים
  • :Node.JS
    • ניווט הבקשות ברשת
    • בניית שרותי REST
    • יצירת דפים דינמיים ע״י Templating Engines
    • מסדי נתונים Object Relational Mapping , MySQL
  • מבוא לספרית React
  • הקמת אתר (Web hosting)
